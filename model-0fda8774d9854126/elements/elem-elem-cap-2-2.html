Release Cycle Management ()
Release Cycle Management controls the release cycle at the Control level, ensuring effective governance of the release process. It progresses through capability increment scoping, model development and semantic validation, release authorisation and publication, from initiation to completion. It receives Cycle Management Feedback from Community & Stakeholder Engagement to inform the release process. It produces Release Approval Documentation for Architecture Publishing, enabling the dissemination of approved releases.
